<h1 align="center">SLUDGE: Sequenced Lineages for Understanding Digester Genomes for Engineering</h1>

<h2>Key Capabilities</h2>
<ul>
  <li>Comparative genomics across digester ecosystems</li>
  <li>Metabolic pathway reconstruction for key microbial lineages</li>
  <li>Community function modeling to inform process optimization and engineering design</li>
</ul>

<h2>ğŸ§¬ Why SLUDGE?</h2>
<p>
  Anaerobic digesters are complex, microbially-driven ecosystems central to sustainable waste management and bioenergy. 
  The diversity and function of many microbial lineages remain poorly understood. SLUDGE bridges this gap by linking 
  genome-level data to system performance, accelerating both basic and applied research in environmental biotechnology.
</p>
